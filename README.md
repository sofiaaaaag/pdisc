El diagrama que hice es sobre un sistema de simulación de mascotas. La primetra clase se llama "sistema de simulación de mascotas", la cual tiene un atributo "raza: Gato". La cual se conecta con otra clase llamada "Gato", que teambien tiene el atributo pelaje; el cual se conecta con una clase llamada Pelaje. A su vez; De vuelta en la clase "Gato", tenemos un metodo Maullar, La clase gato le hereda el metodo maullar a la clase "Gato Montes".
